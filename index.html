    What is Object Oriented JavaScript

    Any programming language updated day by day by so that programming language will be good and be faster programs. when any programmer write program he primary purpose his program fill on security by which any person can’t lost his program so we used object oriented programming language. The second aspect of Object-Oriented Programming is about structuring code when you have multiple levels of blueprints. This is commonly called Inheritance or subclassing. JavaScript is also object oriented programming language.in JavaScript class and Constructor , objects, objects-properties use as object oriented JavaScript. Object-oriented language usually are recognized through their use of classes for creating various objects ,which have similar properties and methods.

    JavaScript is an object-oriented programming language and so a object-oriented programming language can be called object-oriented when this provides developers with at least four basic capabilities to develop.
    1.Encapsulation-: This is the capability for storing related information, whether data or methods, mutually in a single object.
    2. Aggregation-: It is the ability to store one object inside another.
    3. Inheritance-: A class can depend upon another class or number of classes and inherit their variables and methods for some specific use.
    4. Polymorphism-: It is the potential of the concept of OOP for writing one function or method which works in a variety of different ways.

    What is Breakpoint in Bootstrap

    Bootstrap framework of CSS which use Responsive design. Bootstrap says, first of all your website should be Responsive and bootstrap`s main goal of website make mobile friendly. for which Bootstrap has made different-different breakpoints by which use we can make our website responsive and mobile friendly. Bootstrap given us six type of breakpoints whom we can use for different-different Device by which use we can our website Mobile Friendly
    1.Extra extra large(xxl)-: These breakpoints are use to make your website responsive on more devices.to use it, we can use it by writing its class created by bootstrap in your HTML page. To write this class we have to write to as class in our <div> tag Write Methods-: <div class = “xxl”> div > and in this the width of our container is >1400 px .

    2.Extra large(xl)-: we use this breakpoint to make the design of our website and layout the best on a device like laptop , Computer,Tv,Moniter. Write Methods-:<div class = “xl”> div > and in this the width of our container is >1200px.

    3.large(lg)-: in this breakpoints we use to make our layout responsive in a device like a tablet, small laptop. and use after this breakpoint our website become tablet friendly for any tablet user. Write Methods-:<div class = “lg”> div > and in this the width of our container is >992px.

    4.Medium(md)-: in this breakpoints we use to make our layout design responsive in a device like a small lablet, mobiles. and now our website become mobile friendly. Write Methods-: <div class = “md”> div > and in this the width of our container is >768px.

    5.Small(sm)-: in this breakpoints we make our website total mobile friendly . Now days there are any mobile device in this breakpoint we cover the size of them all Write Methods-: <div class = “sm”> div > and in this the width of our container is >576px.

    6.X-Small(none)-: We write this breakpoints because if the design of our website gets spoiled at any of the breakpoint, then it design the layout by taking the full width of the container.to write this , we do not have to use any class ,only all of our grid have to be given to it. If any company wants to make a small device tomorrow, we can also use breakpoint for the size of the device, it is possible to make our layout responsive according to small device.

    Why Use React.js

    React.js is java script library. It is use for building user interfaces and this is Component based library. This library Maintained by Facebook.it is primary goal is to make reason about an interface and its state at any point in time, by dividing the UI into a collection of components. React.js First Time Used in 2011 for Facebook Newsfeed feature.it is created by Facebook Software Engineer Jordan Walke and initial Release to the public was in July 2013. It is used to build a single page application and to make good user experience. Since Component logic is written in JavaScript instead of template. you can easily pass rich data though your app and keep state out of the DOM. React "reacts" to state changes in your components quickly and automatically to rendered the components in the HTML DOM by utilizing the virtual DOM.
    Why Use React.js: Nowdays mostly people use react because there so many open -source platform for making the front-end web application development easier like other framework of JavaScript.

    1. Components
    2. JSX (JavaScript XML)
    3. State
    4. Props

    3.Performance-:react does not offer concept of a built-in container for dependency you . There is no concept of a built-in dependency container in React.
